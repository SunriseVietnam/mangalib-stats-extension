# 📊 MangaStats Tracker for MangaLib

> Расширение для Chrome, которое отслеживает и анализирует статистику тайтлов на MangaLib: оценки, количество читателей, динамику популярности и прогресс.

## ✨ Возможности

- **Автоматический сбор данных** – при посещении страницы манги все данные сохраняются локально
- **Средняя оценка и количество голосов** – сумма всех оценок (10,9,8...1)
- **Статистика по спискам** – сколько человек добавили тайтл в "Читаю", "В планах", "Брошено", "Прочитано", "Любимые", "Другое"
- **Детальная информация** – количество глав, статус, жанры, автор
- **📈 Графики динамики** – визуализация изменений популярности тайтла с возможностью сравнения двух тайтлов
- **➕ Добавление по ссылке** – можно добавить любой тайтл без посещения страницы (поддерживается несколько ссылок сразу)
- **🗑 Управление списком** – удаление отдельных тайтлов или полная очистка
- **💾 Экспорт данных** – выгрузка всей статистики в JSON или CSV
- **🔒 Приватность** – все данные хранятся только локально в вашем браузере
